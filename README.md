# LoRA Asset Prep - LoRA 素材预处理

一款LORA训练素材裁剪缩放工具。包含等比缩放、黑 / 白 / 透明补边，支持自动批处理或手动裁剪（开发中）。

```powershell

pip install -r requirements.txt
python app.py
```
