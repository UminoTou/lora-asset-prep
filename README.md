# LoRA Asset Prep - LoRA 素材预处理

一款 LoRA 训练素材预处理工具：支持多文件夹 / 多张图片（含拖放）汇总后批量处理，等比缩放与黑 / 白 / 透明补边。

## 功能要点

- **素材**：可添加多个文件夹与多个图片路径，或从资源管理器一次性拖入；同一文件 / 文件夹不会重复计入。

